# Gastos — Personal Budget Tracker

A full-stack personal finance app built for solo living in Manila.
AI-powered insights (Peso Buddy), expense tracking, budget limits, and savings goals.

## Tech Stack

- Next.js 14 + TypeScript
- Mantine UI + Charts
- PostgreSQL + Prisma 6
- NextAuth.js (Google OAuth)
- LLaMA 3
